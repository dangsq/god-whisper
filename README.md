# 渊 · Oracle Whisper

一个探索创作过程的交互艺术装置。

## 概念

> 创作是某种通灵的过程——作者接收潜意识的声音，将其转化为作品。

这个项目呈现了一个三元结构：

```
神（潜意识）→ 抽象意象碎片
    ↓
创作者 → 被触发，创作出独立作品
    ↓
观众 → 见证这个过程
```

## 体验流程

1. **召唤** — 点击"召唤"按钮
2. **神谕** — 神发出破碎的抽象意象（如"崩塌的大地 / 铁花"）
3. **创作** — 创作者被触发，写出一段故事/诗/场景
4. **凝视** — 背景中的"眼睛"持续注视着你

## 设计理念

**视觉**
- 极简黑白美学
- 中央的"眼睛"象征被凝视的感觉
- 纯黑背景营造虚空感

**文本**
- 神谕：15-25 字符的抽象意象碎片
- 创作：3-6 句话的独立作品（故事/诗/场景）

**交互**
- 无需用户输入文字
- AI 扮演"神"和"创作者"两个角色
- 用户作为观众见证创作过程

## 技术栈

- **前端**: Vite + Vanilla JS
- **样式**: CSS (无框架)
- **AI**: OpenAI API (或兼容接口)
- **部署**: GitHub Pages

## 本地运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 配置

创建 `.env` 文件：

```env
VITE_API_KEY=your-api-key
VITE_API_BASE_URL=https://api.openai.com/v1
VITE_API_MODEL=gpt-4
```

或在首次打开页面时输入 API 配置。

## 哲学背景

这个项目探讨：

- **创作的本质** — 是理性构建还是潜意识涌现？
- **通灵隐喻** — 作家/艺术家是否在"接收"某种信息？
- **意义的生成** — 从混沌到秩序的过程
- **观看与被观看** — 创作时的自我意识

## 灵感来源

- 自动书写 (Automatic Writing)
- 荣格的集体潜意识
- 神谕与占卜传统
- 现代名人造神现象
